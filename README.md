# TikTacToeWithGPT
ChatGPT API를 활용한 인공지능 상대 틱택톡 게임입니다.

플레이어는 ChatGPT 와 틱택톡 게임을 통해 승/패를 겨룹니다.

현재 버전은 [1.0.0] 입니다. 계속해서 디자인, 추가기능을 업데이트를 진행할 예정입니다.

### 배포링크: https://sungbinlee.github.io/TicTacToeWithGPT/
